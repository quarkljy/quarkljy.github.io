Posted on 2019-05-16 In mysql Views:
使用Dockerfile创建镜像，docker-compose部署主从同步。
1. 前言

将数据库扩展成 主从复制模式，将 读操作 和 写操作 分离开来，多台数据库 分摊请求，从而 减少单库的 访问压力，进而应用得到优化。
